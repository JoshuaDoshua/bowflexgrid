BowFlex Grid

About

  • FlexBox grid system
  • Attribute based
  • Mobile first
  • SASS Mixins
  • Customizable
  • Full FlexBox capabilities **almost
  • Built to handle Horizontal layout

Use default build if...

  • 12 Columns
  • Basic Breakpoints
  • 505 :: 768 :: 1024 :: 1400
  • >2% browser support

Use custom build if...

  • <> 12 Columns
  • Custom breakpoints
  • Custom named breakpoints
  • Advanced browser support
